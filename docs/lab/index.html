파이프라인 테스트 랩

P01(교안 분석) · P02(코드 분석) · P03(인터뷰) — 각 파이프라인의 실제 코드가 매 단계 어떤 프롬프트(또는 파라미터)를 쓰는지 그대로 보고, 편집해서 다시 돌려볼 수 있다. 결과는 팀 DB에 실행자 계정으로 기록된다.

설정 필요 — 자세한 절차는 SETUP.md 참고.

연결 설정

이 탭 안에서만 유지된다(선택 시 sessionStorage) — 서버로 전송되지 않고, DB에도 저장되지 않는다. 결과 DB는 팀 공용으로 이미 연결돼 있음 — 아래 Google 로그인만 하면 본인 실행이 저장된다.
