Hook File 재귀 루프 — 개인화 처방이 실제로 도움이 되는가 (γ, 잠정보고, D121~D127)

세 파이프라인이 만든 진단(취약축)을 학생 개인의 "다음 과제 체크리스트"(Hook File)로 압축해 회차마다 갱신하는 루프. 실학생을 모집하기 전, 이 장치 자체가 오염 없이 k=4회차를 끝까지 도는지부터 에이전트 시뮬레이션(γ, 페르소나 1명 자기대조)으로 먼저 확인했다 — 증거력은 잠정보고 수준이다. n=1 자기대조라 인과를 확정할 수 없고, 병행 대조군(β)이 있어야 "hook 때문"과 "그냥 익숙해짐"을 분리할 수 있다.
